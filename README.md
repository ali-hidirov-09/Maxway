# Maxway - Food Management Platform

**Modern Django web application for food management with user authentication and responsive design.**

## Features

- 🔐 User authentication (login/logout)
- 📊 Dashboard with analytics
- 🍕 Food management module
- 📱 Responsive design
- 🗄️ PostgreSQL database
- 📦 Multi-app architecture

## Tech Stack

**Backend**: Django 5.1.4, PostgreSQL  
**Frontend**: HTML5, CSS3, JavaScript  
**Database**: PostgreSQL + psycopg2  
**Libraries**: Pillow 12.1.0, Asgiref 3.10.0
